# Exam Details and Links
This document gives some basic details about the actual **Prometheus Certified Associate (PCA)** exam, and some important links. Keep in mind that some of the details and links can change over time.

*Last updated: 2/25/2026*

## Exam Details
- Format: Online Proctored
- Types of questions: Multiple choice
- Number of questions: 60
- Duration: 90 minutes
- Language: English
- Credential expiration: 3 years
